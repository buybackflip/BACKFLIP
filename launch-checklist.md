# $BACKFLIP — Launch Checklist

Work through this in order. Don't skip steps.

---

## PRE-LAUNCH

### token
- [ ] Token deployed on [Solana / Base / ETH — confirm chain]
- [ ] Contract address confirmed and saved
- [ ] Liquidity locked (document tx hash)
- [ ] Mint authority renounced (document tx hash)
- [ ] Contract verified on explorer

### listings & tracking
- [ ] DEX listing confirmed (Raydium / Uniswap / etc.)
- [ ] CoinGecko submission filed
- [ ] CoinMarketCap submission filed
- [ ] DEXScreener profile set up (logo, links, description)
- [ ] DEXTools profile set up

### socials
- [ ] Twitter/X account created → @backflipcoin
- [ ] Telegram group live and linked
- [ ] Bio, banner, and pinned post ready
- [ ] First 5 tweets queued and scheduled

### content
- [ ] 3+ backflip videos sourced (with permission or licensed)
- [ ] Logo finalized (transparent PNG, 400x400 minimum)
- [ ] Banner image ready (1500x500 for Twitter)
- [ ] Launch caption ready to post

---

## LAUNCH DAY

- [ ] Post contract address across all channels simultaneously
- [ ] Pin contract + chart link in Telegram
- [ ] Post launch tweet (see captions/twitter-captions.md)
- [ ] Drop first backflip video with $BACKFLIP tag
- [ ] Begin replying to viral backflip content on X
- [ ] Message first wave of influencers (see outreach/influencer-dm.md)
- [ ] Monitor chart — screenshot first green candle and post it

---

## POST-LAUNCH (first 48 hours)

- [ ] Post at least 2x per day on X
- [ ] Engage every reply and comment personally (first 24h)
- [ ] Repost any organic mentions
- [ ] Track which captions are performing — double down
- [ ] Log any influencer responses
- [ ] File for additional CEX listings if volume warrants

---

## NOTES

> momentum compounds fast or dies fast. the first 6 hours matter most.
> have someone watching the chart and someone watching X at all times on launch day.
> don't go dark. presence = trust in the first window.
